# Scala-wasm Spin templates

Spin templates for scala-wasm.

## Install

Install from GitHub:

```sh
spin templates install --git https://github.com/scala-wasm/scala-wasm-spin-templates
spin new -t http-scala-wasm-todo <project>
```

## Templates

- `http-scala-wasm-todo`: HTTP TODO API using scala-wasm and Spin SQLite.
